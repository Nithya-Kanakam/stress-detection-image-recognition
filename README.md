
# 🧠 Stress Detection in IT Professionals using Image Recognition

> A computer vision-based system to detect stress levels in IT professionals using facial image analysis.

---

## 📌 Project Overview

This project focuses on detecting stress in IT professionals using image recognition techniques. 

The system analyzes facial features from images and predicts stress levels, highlighting how computer vision can be applied in mental health monitoring and workplace well-being.

---

## 🎯 Objective

The goal of this project is to:

- Detect stress levels from facial images  
- Explore how visual cues can indicate mental state  
- Build a simple end-to-end system combining machine learning and a web interface  

---

## 🛠️ Tech Stack

- Python  
- OpenCV  
- Django (Web Framework)  
- Machine Learning (Image-based classification)  

---

## 🧠 Methodology

### 1. Data Collection
- Collected facial images representing different stress conditions  
- Organized dataset for training and testing  

### 2. Preprocessing
- Image resizing and normalization  
- Feature extraction using computer vision techniques  

### 3. Model Development
- Trained a classification model to detect stress levels  
- Evaluated performance based on prediction accuracy  

### 4. Deployment
- Built a web interface using Django  
- Users can upload images and receive stress predictions

## ⚙️ How to Run the Project
